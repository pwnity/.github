**[pwnity-cli](https://github.com/pwnity/pwnity-cli)** is a powerful, open-source command-line interface for orchestrating pentesting workflows. It replaces the mental overhead of manual commands and scattered notes with a structured system for managing targets, tools, and findings.

- Standardize your tool execution with reusable templates.
- Automate your command generation with a powerful placeholder engine.
- Centralize your findings with session-based reporting.
